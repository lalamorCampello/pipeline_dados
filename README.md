# Pipeline de Dados

Meu primeiro projeto de processamento de dados com Python.

## Funcionalidades
- Leitura de arquivos JSON e CSV
- União de datasets
- Renomeação de colunas
- Exportação para CSV

## Como usar
1. Clone o repositório
2. Execute o notebook fusao_mercado.ipynb
